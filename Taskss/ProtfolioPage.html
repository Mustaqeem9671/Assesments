FoodMarket Project Overview:

  • During my final semester of BCA, I undertook the development of an innovative project called "FoodMarket." This web application bears similarities to popular platforms like Swiggy and Zomato, presenting a seamless experience for both users and administrators.

    Key Modules:

    User Module:

    Users can log in securely using their credentials. A dynamic and user-friendly interface allows users to browse through a variety of products. The intuitive shopping cart feature enables users to add products and proceed to order placement. Secure online payment functionality ensures a smooth transaction process. Users receive detailed order confirmations and slips upon successful transactions.

    Admin Module:

    Admins possess comprehensive control over the application. Access to all aspects of product management, including addition, modification, and deletion. The ability to oversee and manipulate order statuses, including confirmation, cancellation, and processing. Full user database management, enabling admin to add or remove users as needed.

    Technologies Used:

    Frontend: Developed using React.js for a dynamic and responsive user interface. Backend: Powered by Node.js, facilitating efficient server-side operations. Database: Firebase integration for secure and scalable data storage. Styling: Tailwind CSS for a clean and visually appealing design.
